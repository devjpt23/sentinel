## PDF Export

### Library

Add `@react-pdf/renderer` to `web/package.json`. Lightweight (~50KB), renders React components to PDF, works client-side. No backend involvement.

### Component

Create `web/src/lib/pdf-export.tsx` with a `CompanyReportPdf` document component:

```
Page:
  Header: Sentinel — <Ticker> Report
  Section 1: Company Overview (name, sector, industry, price, market cap)
  Section 2: Health Score + Verdict
  Section 3: DCF Valuation (fair value, upside %, current price)
  Section 4: Risk Summary (score, label, top 3 red flags)
  Section 5: Key Ratios (P/E, P/B, Debt/Equity, Dividend Yield)
  Footer: Generated by Sentinel | Date
```

### Integration

Add "Download PDF" button on `web/src/app/(dashboard)/company/[ticker]/page.tsx`:
- Uses `PDFDownloadLink` from `@react-pdf/renderer`
- Shows loading state while PDF generates
- Triggers browser download on click
- Reuses existing page data — no extra API calls

## Supply Chain Choropleth Map

### Current State

Supply chain page uses a placeholder or missing visualization where Plotly was used on the main branch.

### Solution

Replace with `react-simple-maps` (lightweight, ~15KB) or pure D3 via `d3-geo` + TopoJSON:

- World map with countries colored by supply chain risk score
- Color scale: green (low risk) → yellow → red (high risk)
- Hover tooltip: country name + risk score + exposure level
- Zoom + pan for interactive exploration (optional, nice-to-have)
- Reuses existing `GET /api/data/<ticker>/supply-chain` endpoint data
